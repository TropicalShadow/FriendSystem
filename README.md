# FriendSystem

> PaperMC plugin to introduce `friends` into minecraft (Paper 1.17.1)

## Features

- [x] Core
  - [x] Full Language Configurable
  - [x] Store Playerdata to file
- [x] Friend requests
  - [x] Send & Receive
  - [x] Cancel & Approve
- [ ] GUI
  - [ ] Friend List
  - [ ] Friend Managment
- [ ] Notifications
  - [x] Friend Requests
    - [x] Receive
    - [x] Accepted
  - [ ] Friend
    - [ ] Join
    - [ ] Leave
- [ ] Party System
  - [ ] Force Teleport on world change
